# RESTful API Project

## Description
This project is a simple and well-structured RESTful API built using Node.js and Express.js. It follows a modular architecture using routes, controllers, middleware, and utilities.

## Technologies Used
- Node.js
- Express.js
- UUID
- Express Validator

## Project Structure
- server.js → Main server file
- routes/ → API routes
- controllers/ → Business logic
- middleware/ → Error handling
- utils/ → Helper functions

## How to Run

1. Install dependencies:
npm install

2. Start server:
node server.js

## API Endpoints

### Users
- POST /users/register
- POST /users/login

### Products
- GET /products
- POST /products

### Cart
- GET /cart
- POST /cart

### Orders
- GET /orders
- POST /orders

## Testing
Use Postman to test API endpoints.
